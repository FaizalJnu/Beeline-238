.. _boolode:
   
BoolODE
=======

Overview
--------

The BoolODE package converts a Boolean model to an ODE model and carries out stochastic
simulations.

.. code:: bash

   python src/BoolODE.py --path=data/test_vars.txt --max-time=5 --num-cells 10
